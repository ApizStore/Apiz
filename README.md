## 🎯 What is Apiz Store?

Apiz Store is a modern web application where gamers can:
- 🎮 **Browse Games** - Discover popular games across all platforms
- ✨ **Suggest Games** - Submit your favorite games for others to discover
- 🔍 **Search & Filter** - Find games by name, platform, or genre
- 📥 **Download** - One-click access to PlayStore, Steam, AppStore, and more
- 👍 **Vote** - Suggest the same game multiple times to boost its ranking

---

## 🚀 Features

✅ **Game Suggestions** - Anyone can suggest a game with description and download links

## 📱 How to Use

### **Visit the Store:**
Open your browser and go to:
```
https://ApizStore.github.io/Apiz/
```

### **Search for Games:**
1. Use the search bar to find games by name
2. Click platform buttons to filter (PC, Android, iOS, Console)
3. Browse the game cards

### **Suggest a Game:**
1. Fill out the form with:
   - Game name
   - Description
   - Available platforms
   - Genre
   - Download link (optional)
2. Click "🚀 Submit Suggestion"
3. Your game appears in the store!

### **Download Games:**
Click the 📥 **Download** button on any game card to go directly to the store page.
